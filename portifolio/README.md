# 💻 Portfólio Pessoal - Daniel

Projeto desenvolvido para a disciplina de Programação Front-End.

---

## 📌 Descrição

Este projeto consiste em um website de portfólio pessoal desenvolvido utilizando HTML5, CSS3 e JavaScript.  
O objetivo é demonstrar conhecimentos em desenvolvimento front-end, responsividade, SEO básico e interatividade.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Poppins)

---

---

## 🎯 Funcionalidades

- Layout responsivo (desktop, tablet e mobile)
- Menu de navegação funcional
- Animações e efeitos com JavaScript
- Botão voltar ao topo
- Validação de formulário
- Estrutura otimizada com SEO básico

---

## 📱 Responsividade

O site se adapta a diferentes tamanhos de tela utilizando Media Queries.

---

## ⚙️ Como executar

1. Baixe ou clone o projeto
2. Abra o arquivo `index.html` no navegador
3. Navegue pelas páginas normalmente
